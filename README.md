Order micro service
How to run?
Local build
Prerequisites
Java 8+
Maven 3.6.3
Mongodb 6.0
Steps
> mvn clean package
> mvn spring-boot:run 
Open a browser and access the following URL for swagger
http://localhost:9192/swagger-ui/index.html#/
